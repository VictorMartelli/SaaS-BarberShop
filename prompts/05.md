## Tarefa

- Criar um sheet de cancelamento de reserva que é exibido quando o usuário clica no @app_components\booking-item.tsx.
- A interface deve ser EXATAMENTE a que está no Figma em https://www.figma.com/design/F1N5Gfe5hXEdmoJLJy1S04/SaaS-BarberShop?node-id=78-2076&m=dev.
- Ao clicar no botão de "Cancelar Reserva" a reserva deve ser cancelada.

## Requisitos Técnicos

- Use o Sheet do shadcn/ui.
- Edite a Server Action de cancelar a reserva chamada "cancel-booking" que está em @app_actions\cancel-booking.ts. Ela deve receber o ID da reserva e definir booking.cancelled = true.
- Os dados exibidos no Sheet devem ser os mesmos dados do agendamento clicado.
- A imagem do mapa está em @public\map.png
- Status é confirmado se agendamento é no futuro e finalizado se é no passado.
- Reutilize o componente @app_components\phone-item.tsx para os números de telefone.
