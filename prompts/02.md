# Requisitos e Detalhes Técnicos

- O avatar, nome e e-mail devem ser do usuário logado.
- Botão de início deve levar apenas para a tela inicial.
- Botão de agendamentos deve levar para a página "/bookings" (**NÃO CRIE ESSA PÁGINA AINDA**).
- Os botões de categoria (Cabelo, barba etc.) **NÃO DEVEM FAZER NADA, POR ENQUANTO**.
- Botão de "Sair" deve fazer logout.
- Use o componente Sheet do Shadcn para exibir o componente.
- Esse componente deve ser aberto ao clicar no botão de Menu que está no @app_components\header.tsx.
